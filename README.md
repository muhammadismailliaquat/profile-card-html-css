# Personal Profile Card

A clean and modern profile card built using HTML and CSS.

## Features

- Circular profile image
- Name, designation, and short bio
- Skills section
- Social media links
- Two call-to-action buttons
- Hover effects on card and image

## Technologies Used

- HTML5
- CSS3
